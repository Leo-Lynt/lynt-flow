## Quick Start

<Steps>
  <Step title="Install LyntFlow">
    Get started by installing the package
    ```bash
    npm install @LyntFlow/core
    ```
  </Step>

  <Step title="Create Your First Flow">
    Build a simple flow to add two numbers
    ```json
    {
      "nodes": [
        { "id": "1", "type": "input", "data": { "value": 5 } },
        { "id": "2", "type": "input", "data": { "value": 3 } },
        { "id": "3", "type": "add" }
      ],
      "connections": [
        { "from": "1", "to": "3", "handle": "a" },
        { "from": "2", "to": "3", "handle": "b" }
      ]
    }
    ```
  </Step>

  <Step title="Execute & Get Results">
    Run your flow and see the magic happen
    ```typescript
    import { FlowEngine } from '@LyntFlow/core';

    const engine = new FlowEngine();
    const result = await engine.execute(flow);
    console.log(result); // Output: 8
    ```
  </Step>
</Steps>
